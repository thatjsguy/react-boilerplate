# React Boilerplate Demo

The goal is to build it lightweight and mostly un-opinionated.

## Features

- WebPack 2
- Express 4
- ES6
- React
- Redux
- react-router-redux
- CSS Modules - SASS (TODO)
- Universal rendering

## Installation

1. Ensure node version from `.nvmrc` is installed
2. `npm install`
3. `npm start`
